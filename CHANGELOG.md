# Changelog

## v1.0.0

- Initial public release of Telegram Music Bot (Go).
- Search songs in Telegram and download to a local server path.
- Support download quality selection and per-chat default quality.
- Support configurable token file path, source order, and source API base URL.
- Support multi-source fallback via the GD music API.
- Support Docker Compose and Debian 13 systemd deployment.

## Unreleased
